TIPOGRAFIA AVANÇADA


Este é o curso mais avançado de tipografia para Web que existe no Brasil. Qualquer pessoa interessada no assunto pode fazer o curso, pois todos os assuntos serão abordados do zero. Vamos aprender as classificações tipográficas — Serifadas, Sem Serifas, Script e Mais — e como utilizá-las na web. Também veremos como utilizar as principais variáveis tipográficas que são a Altura de Linha, Largura do Texto, Escala do Tipo, Cores e mais.

Tipografia é um dos assuntos mais importantes para qualquer Web Designer, já que 95% de toda a web é composta apenas por textos. Manipular textos é fácil o difícil é fazer da forma correta. Com este curso você irá melhorar em 80% qualquer projeto que você já tenha feito ou ainda irá fazer. E tudo isso de forma bem fácil, sem necessidade de nenhum talento, basta interesse no assunto.
